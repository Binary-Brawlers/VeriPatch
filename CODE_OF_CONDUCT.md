# Code of Conduct

## Purpose

This project exists to build great software together. This code of conduct keeps collaboration productive and respectful.

## Expected Behavior

- Keep discussions focused on technical merit
- Give and accept constructive feedback on code, design, and ideas
- Respect other contributors' time — come prepared, be concise, search before asking
- Credit others' work and contributions
- Assume good intent in code reviews and discussions
- Help newcomers get oriented in the codebase

## Unacceptable Behavior

- Personal attacks, insults, or hostile language in any project space
- Sustained disruption of discussions or reviews
- Publishing others' private information without consent
- Spam, self-promotion, or off-topic content in project channels
- Deliberately wasting maintainers' time (e.g. bad-faith issues, troll PRs)
- Harassment of any kind

## Scope

This applies to all project spaces: issues, pull requests, discussions, and any channel where you represent the project.

## Enforcement

Violations can be reported to the project maintainers. Reports will be reviewed promptly. Maintainers may warn, temporarily ban, or permanently remove participants who violate this code of conduct.
